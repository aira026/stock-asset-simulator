# 📈 零股資產試算器

> 設定你的股票參數與每月投入，加入任意單月特別注資，即時試算資產達標時間與複利組成。

🔗 **線上使用**：[aira026.github.io/stock-asset-simulator](https://aira026.github.io/stock-asset-simulator/)

---

## ✨ 功能特色

- 📊 設定目前持股數與當前股價
- 💰 設定每月固定投入金額
- 📈 自訂年化報酬率（含股息再投入）
- 🎯 選擇目標金額（100萬 / 300萬 / 500萬 / 1000萬 / 財富自由 2100萬）
- 🗓️ 加入單月特別注資（年終獎金、還款回補等）
- ⚡ 即時顯示預估達標月份與提早幾個月達標

---

## 🚀 使用方式

直接開啟瀏覽器訪問 GitHub Pages，或 clone 後本地開啟：

```bash
git clone https://github.com/aira026/stock-asset-simulator.git
cd stock-asset-simulator
open index.html
```
